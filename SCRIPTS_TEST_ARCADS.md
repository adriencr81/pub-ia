# Scripts de test Arcads AI — 3 secteurs

**Durée cible:** 30-40 secondes chacun (= ~85-100 mots)  
**Tonalité:** Conseil entre pairs, pas de vente agressive  
**Format:** Prêt copy/paste dans Arcads AI

## 1️⃣ NUTRITION — Complément Alimentaire

### Script Test #1 : Engagement problem

**Avatar :** Femme 25-35, casual gym/fitness vibe  
**Durée :** 35 sec  
**Angle :** "Pourquoi tes créas nutri passent inaperçues"

---

```
[HOOK - 0-3 sec]
"Tu tournes combien de pubs par mois pour tes suppléments ? 
15 ? 20 ?
Et le vrai souci c'est pas le budget—c'est qu'elles se ressemblent toutes."

[PROBLEM - 3-18 sec]
"Tes audiences ont vu mille fois 'complément naturel' et 'santé' avec les mêmes angles.
Le scrolling est devenu automatique.
Même quand tu changes juste les couleurs, l'engagement suit pas."

[SOLUTION - 18-32 sec]
"Ce qu'on fait nous c'est simple : 
5 angles COMPLÈTEMENT différents en 72h.
Pas juste des variantes—vraiment différents.
Un testé par un mec, un par une fille, un par un athlète, un par un parent.
Chacun a son hook."

[CTA - 32-35 sec]
"On a testé ça avec 3 marques déjà. Les résultats : on les montre sur call.
Calendly ici. On peut tester mardi. ↗️"
```

**Word count:** 95 words ✅

---

### Script Test #2 : ROI problem

**Avatar :** Femme 25-35, casual, kitchen/lifestyle background  
**Durée :** 38 sec  
**Angle :** "Ton budget ads est mal utilisé"

---

```
[HOOK - 0-3 sec]
"Une vraie question : 
Tes 5 dernières créas nutritions, elles ont sortie combien de conversions ?"

[PROBLEM - 3-20 sec]
"Je parie que 3 sur 5 t'ont déçu.
Pas qu'elles étaient mauvaises—juste que tu les créais vite.
Résultat : des vidéos génériques qui font bruit de fond sur le feed."

[SOLUTION - 20-35 sec]
"Nous on change le game : au lieu de créer vite et espérer,
on crée 5 vidéos DIFFÉRENTES et on teste.
On utilise ton produit, TES clients existants,
et on split test par persona."

[CTA - 35-38 sec]
"Déjà fait pour 3 marques nutrition. 
Appel découverte jeudi matin ? Calendly. ↗️"
```

**Word count:** 98 words ✅

## 2️⃣ COSMÉTIQUES — Skincare / Beauté

### Script Test #1 : Trust problem

**Avatar :** Femme 30-40, natural makeup, home background  
**Durée :** 36 sec  
**Angle :** "Comment construire la confiance avec des créas diverses"

---

```
[HOOK - 0-3 sec]
"Les créas beauty qui convertissent vraiment ? 
Elles viennent jamais d'une agence.
Elles viennent de quelqu'un qui teste vraiment le truc."

[PROBLEM - 3-18 sec]
"Tes prospects voient des vitrines polies depuis 3 ans.

Pas des modèles—des profiles qu'tes clients reconnaissent.
Problèmes peau différents : acné, sensibilité, sécheresse, matité.
Chacune teste vraiment."

[CTA - 33-36 sec]
"Résultats visibles sur les 2 marques qu'on a aidées.
Appel vendredi pour les voir ? ↗️"
```

**Word count:** 96 words ✅

## 📋 Checklist avant envoi à Arcads

Pour chaque script :

- [ ] Longueur: 30-40 sec (85-100 words)
- [ ] Pas de "achète maintenant" ou termes salesy
- [ ] Hook en première phrase (0-3 sec)
- [ ] Problem relatable pour persona cible
- [ ] Solution crédible (pas magique)
- [ ] CTA spécifique (Calendly + jour/heure suggestion)
- [ ] Tonalité: conseil entre pairs (pas prédicateur)
- [ ] Personnage avatar clairement identifié
- [ ] Aucune exagération (véracité = clé)

---

## 🎬 Instructions Arcads pour chaque script

### Workflow copy-paste

1. **Créer nouveau project** → Sélectionner secteur (Food/Beauty)
2. **Copier script entier** (HOOK + PROBLEM + SOLUTION + CTA)
3. **Coller dans champ Script**
4. **Sélectionner avatar** (voir suggestions au-dessus)
5. **Voice settings :**
   - Langue: Français
   - Débit: Normal (pas trop rapide)
   - Émotion: Conversational (pas corporate)
6. **Générer** → Attendre 90-120 sec
7. **Évaluer** :
   - Avatar naturel ? (pas robote)
   - Lip-sync correct ?
   - Durée ~30-40 sec ?

---

## 💡 Variations possibles

Une fois la vidéo test générée, tu peux itérer rapidement sur :

1. **Même script, avatar différent** (plus jeune / plus âgé)
2. **Même angle, script raccourci** (25-30 sec au lieu de 35)
3. **Même angle, émotion différente** (sérieux vs relaxé)

**Timing:** Chaque itération = ~3-5 min Arcads + 2 min éval = rapide.

## 📊 Post-production CapCut (après Arcads)

### Assets à avoir PRÊT avant montage

**Nutrition :**
- 2-3 clips du complément (produit en main, ouverture, prise)
- Background : gym / cuisine / extérieur
- Si possible : testimoniaux client (images)

**Cosmétiques :**
- Produit en action (appliqué sur peau)
- Avant/après (optional mais puissant)
- Ingredient close-up (si pertinent)

**Boissons :**
- Boisson versée / dégustation
- Moment du jour (matin/après gym/soir)
- Packaging hero shot

### Template CapCut simple

```
[0-5 sec]    Intro
             Asset produit + texte headline

[5-35 sec]   Avatar Arcads (son complet)
             Arrière-plan: assets produit en montage

[35-40 sec]  CTA
             Texte: "Voir les résultats"
             Avatar bonus ou produit gros plan
```
