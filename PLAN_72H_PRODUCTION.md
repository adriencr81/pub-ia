## 🎬 JOUR 2 — Montage CapCut (6-8h)

### Matin (9h-13h) : Montage principal

**Setup CapCut :**
1. **Installer** : https://www.capcut.com (gratuit desktop version)
2. **Créer nouveau projet**
   - Nom : `Nutrition_test_arcads_v1`
   - Format : 1080p (9:16 vertical)
   - FPS : 30
3. **Importer la meilleure vidéo Arcads**
   - Drag & drop sur timeline

**Workflow montage (33 min):**

```
[00-05 sec] INTRO
├─ Importer photo produit (asset client) en arrière-plan
├─ Ajouter texte overlay :
│  - Font: Bold, white, large
│  - Texte: "Tes créas nutri passent inaperçues ?"
│  - Position: Center
│  - Durée: 4 sec
├─ Ajouter transition Dissolve (1 sec)
└─ FX: Brightness +10% pour produit visible

[05-32 sec] AVATAR ARCADS + B-ROLL
├─ Vidéo Arcads = piste principale (son)
├─ Ajouter clips produit pistes visuelles :
│  - 0-8 sec : Produit ouverture (clip client)
│  - 8-15 sec : Produit en action (photo + zoom)
│  - 15-22 sec : Client testimoniaux (image si dispo, sinon b-roll)
│  - 22-32 sec : Call-to-action visuelle (produit hero + avatar mini)
├─ Transitions entre clips : Dissolve 0.5sec
├─ Sous-titres AUTO (CapCut génère du son Arcads)
│  ├─ Ajouter via "Captions" → Auto-generate
│  ├─ Style: Bold outline (lisibilité)
│  └─ Position: Bottom
└─ Music: NONE (son Arcads uniquement)

[32-38 sec] CTA (Call-to-action)
├─ Arrêter avatar Arcads
├─ Texte overlay:
│  - "Voir les résultats →"
│  - Calendly link en commentaire IG post
│  - Position: Center 60% écran
├─ Produit gros plan background (2-3 sec)
└─ Transition fade out
```

**Checklist montage :**
- [ ] Sous-titres lisibles (contraste white/black outline)
- [ ] Produit visible >= 40% du temps
- [ ] Pas de silence/blanc (son du début à fin)
- [ ] Transitions lisses (pas abrupt)
- [ ] Durée finale : 35-40 sec ✅
- [ ] Format : 1080 × 1920 (9:16) ✅
- [ ] FPS 30, H.264 codec ✅

**Temps montage :** ~45 min (première vidéo, moins rapide après)
