## 🚨 RED FLAGS à corriger avant livraison

| Issue | Fix time | Action |
|-------|----------|--------|
| Avatar parle trop vite | 5 min | Réduire débit Arcads, re-générer |
| Sous-titres illisibles | 5 min | Font 50pt + black outline 5px |
| Produit peu visible | 10 min | Augmenter durée B-roll à 30 sec min |
| CTA floue | 5 min | Ajouter texte "Voir résultats →" |
| Watermark CapCut visible | N/A* | Ignorer si test gratuit, mention "upgrade for logo-free" |
| Audio mute spots | Re-export | Vérifier volume Arcads audio, re-export |

*CapCut gratuit = watermark acceptable pour MVP, Starter plan le retire

## 🎯 Success metrics (Know when to pivot)

### VERT (Tout va bien → Continue)
- [ ] Avatar crédibilité: ⭐⭐⭐⭐⭐ (oublie que c'est IA)
- [ ] Script engagement: ⭐⭐⭐⭐ (client reconnaît pain point)
- [ ] Production quality: ⭐⭐⭐⭐ (comparable agence "vraie")
- [ ] Client réaction: "Wow, on peut faire ça en 24h ?" (positive shock)

### ORANGE (Adjust)
- [ ] Avatar "OK mais un peu plastique" → Tester autre avatar (5 min)
- [ ] Client feedback: "Script pas assez hard-sell" → Ajuster tonalité (30 min redo)
- [ ] Production feels "slightly amateurish" → Améliorer sous-titres + transitions (10 min)

### RED (Pivot immediately)
- [ ] Avatar "Trop clairement robot" → Tester HeyGen Plan B (1-2 jours)
- [ ] "Too synthétique pour notre audience" → Hybrid approach (trouver vrais clients pour avatar)
- [ ] "5 jours pas assez court" → Réduire variantes (livrer 3 au lieu de 5)
