---

## 📋 Préambule crucial

### ⚠️ Limitation Arcads à intégrer
**Arcads NE PEUT PAS :**
- Tenir/montrer les produits physiquement
- Faire des gestes précis (unboxing, démonstration)
- Afficher des détails produit en gros plan

**Arcads PEUT :**
- Avatar UGC ultra-convaincant qui parle
- Créer du hook émotionnel + problème
- Générer 5 variantes d'un même script en ~10 min

**→ Conséquence :** Ton workflow doit être **Avatar Arcads (hook + problème + CTA) + CapCut (montage produit réel + transitions)**

---

## 🚀 Workflow Production Standard (72h pour 5 vidéos)

```
Client signe → brief 10 min
         ↓
Jour 1 (8h)
├─ Analyse concurrence (Meta Ad Library)
├─ Rédiger 5 scripts (variantes angles)
└─ Approuver scripts avec client

Jour 2 (6h)
├─ Générer 5 vidéos Arcads (avatar UGC)
├─ Télécharger + organiser assets clients
└─ Montage CapCut (produit réel + avatar)

Jour 3 (2h)
├─ Finitions (sous-titres, transitions)
├─ Export multiformat (Reels, TikTok, Insta)
└─ Livraison + tutoriel client

TOTAL: ~16h travail effectif
```

## 🎬 Étapes détaillées : Créer 1 vidéo test

### Étape 1 : Préparer le script (15 min)

#### Structure HOOK + PROBLEM + SOLUTION + CTA

**Longueur :** 30-45 secondes MAX (= ~80-100 mots)

**Anatomie :**
```
[0-3 sec] HOOK — Attention immédiate
         "Tes créas nutritions convertissent pas ? Voici pourquoi..."

[3-15 sec] PROBLEM — Pain point client cible
          "Tout le monde dit 'complément sain' mais c'est generic..."

[15-25 sec] SOLUTION — Toi (ton angle de production)
           "Nous on teste 5 angles DIFFÉRENTS en 72h. 
            Objectif: trouver celui qui EXPLOSE en CTR"

[25-30 sec] CTA — Urgent + spécifique
           "On l'a déjà testé pour 3 marques. 
            Calendly ici pour voir les résultats → [lien]"
```

**Ton vocal :** Détendu, pas de vente = conseil entre pairs ("Hey, j'ai trouvé un truc")

---

### Étape 2 : Sélectionner l'avatar Arcads (5 min)

#### Options
1. **UGC Selfie** (RECOMMANDÉ)
   - Style : iPhone selfie frame
   - Effet : Hyper authentique, hyper crédible
   - Démo mentale : "C'est une fille qui teste le produit"
   
2. **Standing avatar**
   - Style : Personne debout parlant
   - Effet : Plus professionnel, moins UGC
   - Éviter pour MVP (moins convertit)

#### Critères sélection avatar
- **Genre :** Match audience cible
  - Femme 25-35 = Nutrition, Cosmétiques
  - Peu importe (neutre) = Boissons, Tech
  
- **Vibe :** "Real person, not corporate"
  - Chercher avatars avec : casual dress, natural makeup, casual background
  - Éviter : costumes, studios professionnels

- **Voix :** 
  - Démos rapides français = mettre son français (accent Québec OK)
  - Sinon anglais avec sous-titres FR

---

### Étape 3 : Générer la vidéo Arcads (3 min)

**Workflow UI Arcads :**

1. **New Project** → Sélectionner secteur (Food/Beauty/etc)
2. **Script** → Copier/coller ton texte
3. **AI Actor** → Chercher & preview avatar sélectionné
4. **Voice** → Tester accent/débit
5. **Generate** → Attendre ~90-120 sec
6. **Download** → Vidéo en MP4

**Attendre résultat et évaluer :**
- ✅ Avatar parle naturellement ?
- ✅ Lip-sync correct ?
- ✅ Pas de bugs/glitches ?
- ✅ Durée respectée (~30-40 sec) ?

---

### Étape 4 : Montage CapCut + Assets client (30 min pour 1 vidéo)

#### Template workflow
```
[0-5 sec]   Intro dynamique
            Asset produit (photo/clip) + avatar en picture-in-picture

[5-25 sec]  Avatar Arcads (son entier)
            Arrière-plan : produit en montage (clips client)

[25-35 sec] CTA visuelle
            Texte overlay + produit gros plan + avatar mini coin
```

#### Instructions CapCut
1. **Importer vidéo Arcads** (piste principale)
2. **Ajouter assets produit client** (picture-in-picture ou arrière-plan)
3. **Ajouter sous-titres auto** (CapCut génère auto du son)
4. **Transitions** : Dissolve simple (pas d'effet "flying text")
5. **Texte overlay** : 
   - Headline (3 sec) : Problem ou hook
   - CTA (2 sec) : "Voir les résultats" + calendly link
6. **Couleurs/Branding** : Si client a palette couleur = ajouter
7. **Export** : 
   - 1080p vertical (Reels/TikTok)
   - 1080p square (Insta Feed)

**Export settings CapCut :**
```
Résolution: 1080p
Ratio: 9:16 (vertical mobile)
FPS: 30
Codec: H.264
```

## 📊 Checklist Vendable (avant livraison client)

### Critères qualité vidéo
- [ ] Avatar semble vrai (pas uncanny valley)
- [ ] Voix naturelle, débit bon (pas robot)
- [ ] Lip-sync exact (pas en retard)
- [ ] Pas de glitches/bugs visuels
- [ ] Transitions lisses (pas saccadé)
- [ ] Durée 30-40 sec (optimal Meta)
- [ ] Produit visible >= 40% du temps
- [ ] CTA claire en finale (+ link/bouton)

### Critères persuasion
- [ ] Hook accroche en 2-3 sec (attention viewer)
- [ ] Problem relatable (client cible se reconnaît)
- [ ] Solution crédible (pas de "magic bullet")
- [ ] CTA urgent + specific ("demain les slots comblent" ou similar)
- [ ] Pas de vente agressive (conseil = plus puissant)

### Critères Meta
- [ ] Format portrait 9:16 (vertical = prio Meta 2026)
- [ ] Sous-titres auto-générés (accessibilité)
- [ ] Pas de music (ou copyright libre)
- [ ] Produit identifiable (pas flou)
- [ ] Avatar crédible (pas "trop IA")

## 📝 Notes techniques Arcads

### Longueur script optimal
- Min: 20 secondes (too short, impression cheap)
- Max: 50 secondes (attention viewers chute)
- **Cible: 30-40 secondes** = temps optimal Meta/TikTok

### Multilang
- FR = à faire en FR directement (meilleure crédibilité)
- anglais + sous-titres = acceptable si avatar anglophone

### Erreurs courantes à éviter
1. **Script trop salesy** ("Achète MAINTENANT") → viewers scroll
2. **Avatar parlant trop vite** → incompréhensible, impression bot
3. **Arrière-plan trop "corporate"** → tue le UGC vibe
4. **Pas de produit visible** → clients disent "je vois rien"
5. **CTA floue** ("contacte nous") → zero conversion vs specific calendly link
