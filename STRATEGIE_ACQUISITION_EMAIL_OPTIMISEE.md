## 💡 **Stratégie Pattern Recognition**

### Si Hunter/RocketReach ne trouvent rien:

**Étape 1: Scraper 2-3 emails connus**
```
1. Aller sur site web de boîte
2. Trouver 2-3 noms (fondateurs, employés visibles)
3. Chercher leurs emails publiquement (Twitter bio, article, etc.)
4. Identifier pattern:
   - Fondateur: "marie.dupont@respire.fr"
   - Employée 2: "thomas.martin@respire.fr"
   → Pattern: firstname.lastname@domaine
```

**Étape 2: Construire email probable**
```
Décideur cherché: "Sophie Bernard" @ Respire
Pattern trouvé: firstname.lastname@domaine
Email probable: sophie.bernard@respire.fr
```

**Étape 3: Valider avant envoyer**
```
Utiliser Clearbit Reveal:
1. Allez sur site Respire
2. Clearbit détecte employés qui visitent
3. Si sophie.bernard@ apparaît → Email valide
4. Sinon → Utiliser secondaire pattern (f.lastname@, firstname.l@)
```
