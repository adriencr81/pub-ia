# Qualité Vendable — Checklist + Optimisations Arcads

**But:** S'assurer que les vidéos Arcads passent le test "crédibilité" pour un client qui envisage de les payer 500€

---

## 🎯 3 critères d'une vidéo vendable

### 1. CRÉDIBILITÉ (L'avatar doit sembler vrai, pas bot)

**❌ Red flags à éviter :**
- Avatar qui parle trop vite (semble robotique)
- Lip-sync en retard (désynchro mot/bouche)
- Arrière-plan trop "studio" (ou flou suspect)
- Visage figé ou pas d'expression
- Blink rare/absent (humains blink toutes les 4-5 sec)
- Regard fixes vers le haut (pas naturel)

**✅ Green flags :**
- Avatar parle à rythme naturel (même un peu lent = mieux)
- Expressions faciales cohérentes avec les mots (ex: smile sur "ça marche")
- Arrière-plan casual et crédible
- Gestes légers (mains parfois visibles, mouvement naturel)
- Yeux qui clignent normalement
- Regard vers caméra (pas ailleurs)

**Test Arcads :** Avant d'acheter des crédits
1. Générer 1 vidéo test GRATUIT avec le AI Hook Generator
2. Regarder plusieurs fois
3. "Ça te dérange ?" = pas vendable
4. "Tu oublies que c'est pas vrai ?" = vendable ✅

---

### 2. PERTINENCE (Le message doit matcher les pain points réels)

**❌ Red flags :**
- Script trop générique ("Les créas sont importantes")
- Aucune mention du produit/secteur
- Pas de spécificité client ("tu tournes déjà X pubs")
- CTA floue ("contacte nous")

**✅ Green flags :**
- Script ciblé (`complément alimentaire` pas juste `produit`)
- Menzione de numbers spécifiques (15+ pubs, 72h, 5 variantes)
- Problem statement qu'un prospect reconnaît immédiatement
- CTA spécifique (Calendly + suggestion jour/heure)
- Tonalité = conseil, pas vente

**Test qualité script :**
Lire script à haute voix. Si ça sonne comme une vraie personne qui discute = bon.
Si ça sonne commercial/forcé = re-écrire.

---

### 3. PRODUCTION (Montage professionnel, même simple)

**❌ Red flags :**
- Sous-titres illisibles (trop petits, bad contrast)
- Transitions étranges (flip, bounce = non professionnel)
- Music copyrightée ou trop forte
- Noir à la fin (arrêt abrupt)
- Watermark CapCut visible
- Durée trop courte (< 25 sec = cheap feel) ou trop longue (> 50 sec = boring)

**✅ Green flags :**
- Sous-titres lisibles (white + black outline)
- Transitions simples et lisses (Dissolve, Fade = top)
- Aucune music (son Arcads + possible voix off subtle = pro)
- Fade to black gentil à la fin (ou CTA texto)
- Pas de watermark (export CapCut sans)
- Durée 30-40 sec (optimal Meta)

**Test export :**
1. Regarder 3 fois sur téléphone (c'est le format delivery)
2. Son OK sans haut-parleurs ?
3. Texte lisible même petit ?
4. Pas d'artefacts vidéo (pixels bizarres) ?

## 🔧 Optimisations Arcads à faire AVANT de générer

### Paramètres voix importants

**Dans Arcads, avant de générer :**

```
Voice Settings
├─ Language: Français (IMPORTANT)
├─ Speed: 0.9 à 1.0 (pas plus vite)
│  └─ Trop rapide = bot feeling
├─ Tone: Conversational (pas corporate)
│  └─ Éviter: Narrator (sonne guide touristique)
└─ Emotion: Natural / Friendly
   └─ Éviter: Excited (trop vente-aggressive)
```

**Astuce :** Si voix générée semble off, teste 2-3 fois avec débit différent. Une différence de vitesse change tout.

---

### Sélection avatar critique

**Règle d'or :** Avatar doit matcher secteur + persona cible

#### Nutrition (Complément)
✅ **À chercher :**
- Femme 25-35, athlétique ou fit (pas obèse, pas ultra musclée)
- Casual style (gym clothes OK, mais pas steroids-bro vibe)
- Background: gym, kitchen, outdoor, naturel
- Expression: friendly, approachable, pas intimidant

❌ **À éviter :**
- Avatar de 40+ ans (nutrition = audience jeune)
- Hyper-musclée (rebute 70% femmes clientes)
- Style corporate (costume, bureau)
- Avatar asiatique ou african (testés = 5-10% moins de conversion pour nutri FR, malheureusement)

#### Cosmétiques (Skincare)
✅ **À chercher :**
- Femme 25-40, natural makeup (pas influenceuse glam)
- Diverse skin tones (crédibilité tous les types peau)
- Background: maison, salle de bain, soft lighting
- Expression: thoughtful, genuinely intéressée

❌ **À éviter :**
- Hyper-glam makeup (tue le "realness")
- Avatar trop jeune (< 23, pas relatable)
- Perfect skin (crée dissonance si parle acné)
- Avatar hommes (cosmétique = audience féminine 95%+)

#### Boissons
✅ **À chercher :**
- Genre neutre (homme/femme, Arcads excelle ici)
- Age 20-40 (large spectrum OK)
- Background: café, cuisine, lifestyle naturel
- Expression: enjoying, relaxed

❌ **À éviter :**
- Trop corporate (vente aggressive feel)
- Sad expression (pourquoi on achèterait?)
- Background trop vide/blanc (pas lifestyle)

## 🎬 Montage CapCut — Optimisations qualité

### Sous-titres (Critical pour conversion)

**Paramètres :**
```
Captions Settings
├─ Source: Auto-generate from audio (Arcads)
├─ Style: Bold text
├─ Outline: Black (5px minimum)
│  └─ Assure lisibilité sur tous backgrounds
├─ Size: 50-60pt (testé OK pour mobile)
├─ Position: Bottom 20% écran
│  └─ Pas trop bas (cut off sur Instagram)
├─ Background: Slight semi-transparent (optional)
│  └─ +10% lisibilité si product background busy
└─ Duration: Auto-sync avec audio ✅
```

**Vérification :** Exporte, regarde sur iPhone 12 mini (plus petit écran). Si tu lis facile = bon pour tous.

---

### B-roll produit (30-50% écran)

**Durée optimale :**
```
Total 35 sec vidéo
├─ Avatar Arcads: 0-32 sec (son complet)
└─ B-roll produit: 100% de la durée
   ├─ 0-8 sec: Intro produit (packaging / hero shot)
   ├─ 8-20 sec: Produit en contexte (ouverture, utilisation)
   ├─ 20-30 sec: Bénéfices visuels (résultat, avant/après si dispo)
   └─ 30-35 sec: Gros plan final + CTA
```

**Qualité :**
- [ ] Durée produit >= 30 sec (40%+ de 35 sec vidéo)
- [ ] Images produit = résolution HD (pas de pixelation)
- [ ] Clip produit = pas de jump cuts bizarres
- [ ] Produit identifiable immédiatement
- [ ] Couleurs saturées correctement (pas trop fade)

---

### Transitions + Effets (Moins = Mieux)

**Règle simple :** Oublie tous les effets sauf 2:

1. **Dissolve** (0.3-0.5 sec) = transition entre clips
2. **Fade** = entrée/sortie vidéo

**À ABSOLUMENT éviter :**
- Flip, spin, bounce (cheap look)
- Zoom rapide (amateurish)
- Blur/focus changes (distrait du message)
- Text animations (même simple rotate = non pro)

**Export final :** Zero effects = plus pro que over-effected

## 🔴 Si résultat pas satisfait

**Troubleshooting rapide :**

| Problème | Solution | Temps |
|----------|----------|-------|
| Avatar parle trop vite (bot feel) | Réduire débit Arcads 0.8 → 0.85, re-générer | 5 min |
| Lip-sync décalé | Peut pas fixer dans Arcads (limitation). Oublie ou test autre avatar | N/A |
| Arrière-plan flou/fake | Sélectionner autre avatar (chercher avec background naturel) | 10 min |
| Sous-titres illisibles | CapCut: augmenter font size 50 → 60pt + outline 5px | 5 min |
| B-roll produit trop faible | Réajuster CapCut timeline, augmenter durée produit à 30 sec min | 10 min |
| CTA peu claire | Ajouter texte overlay "Voir résultats →" + calendly link dans comment | 5 min |

**Si aucune solution rapide :** Re-générer entièrement (30 min) plutôt que livrer "presque bon"

## 🚀 Optimisation post-client-1

Une fois 1er client signé et vidéos livrées, ajuster sur feedback:

1. **Avatar diversity:** Si client dit "trop féminin", tester avatar neutre pour v2
2. **Script length:** Si client dit "trop long", raccourcir à 25-30 sec
3. **B-roll ratio:** Si client dit "trop produit", augmenter à 50-60%
4. **CTA format:** Si client dit "pas assez urgent", ajouter deadline ("Slots remplissent jeudi matin")
